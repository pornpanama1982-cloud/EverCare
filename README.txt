===========================================
 🛡️ EverCare Smart System - Blockchain 
===========================================

การติดตั้งและทดสอบ:
1. แตกไฟล์ ZIP ทั้งหมดไว้ในโฟลเดอร์เดียวกัน
2. นำโฟลเดอร์ไปเปิดในโปรแกรม VS Code
3. สำคัญ*: กรุณาใช้ส่วนขยาย 'Live Server' ในการรันไฟล์ index.html เพื่อให้ไฟล์ module ทำงานเชื่อมหากันได้

ฟีเจอร์สำคัญ:
- โค้ดถูกแยกเป็นไฟล์ .html (แต่ละหน้า), .css และ .js อย่างสมบูรณ์
- ข้อมูล Blockchain จะยังคงอยู่แม้เปลี่ยนหน้า (ใช้ LocalStorage เชื่อมต่อ)
- มีระบบเชื่อม Firebase Google Authentication ในหน้า script.js
